Basics of the Web and HTML

Major components of the Web

  • End User
  • End User's Computer and Browser
  • The Internet
  • Servers
  • Requests are sent over the web from the End User to the Servers using
    the HTTP protocol.
HTML
Hyper Text Markup Language. The glue that holds a web page together.
  • Text content
  • Mark up
  • References to other documents
  • Links to other pages
Hyperlink
Links pages together

HTML Markup

Elements
The building blocks that, when combined, create the structure of a web page.
  • (tag attribute="define a property of the element") contents (/tag)
  • Void Tags
  • Inline vs Block
  • span (inline) vs div (block). Each can take CSS class attributes that are used to add styles to documents.
  • HTML Element and Tag Reference

Structure of an HTML Document

Doctype
The Doctype must be the first thing in an HTML document. It tells browsers what kind of document
the web page is, so that it knows how to render it. Think of it like a file extension for web content.
HTML
This is the root from which all other HTML elements grow. The opening tag comes
just after the Doctype declaration and the closing tag comes at the end of the
document.
Head
The head contains metadata, javascript, and references to CSS style sheets.
Body
Contains all the contents of an HTML document.
